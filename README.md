# CodeAlpha_SQLInjectionProtection

## Description

This project was developed as part of the CodeAlpha Cyber Security Internship. It is a Python Flask web application that helps detect and prevent SQL Injection attacks using secure authentication and parameterized database queries.

## Features

* Secure User Registration
* Secure Login System
* SQL Injection Detection
* SQL Injection Test Lab
* Attack Logging
* Dashboard Monitoring
* SQLite Database Integration
* Responsive HTML and CSS Design

## Technologies Used

* Python 3
* Flask
* HTML5
* CSS3
* JavaScript
* SQLite
* Git & GitHub
* Kali Linux

## Installation

Install Flask:

```bash
sudo apt install python3-flask -y
```

or

```bash
pip install Flask
```

## Run the Project

```bash
python3 app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
